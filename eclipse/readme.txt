# What's about

This eclipse.zip contains the Eclipse IDE for C/C++ project. 

## What it needs

It needs Java JRE and C/C++ compiler. 

Open a cmd console, type the following command to check if Java is available on your machine. 

java -version 

If Java is not available, you need to install Java. 

Type the following command to check if C compiler is available on your machine.

gcc --version

If C compiler is not available, you need to install a C compiler.

## Install Eclipse

Unzip eclipse.zip to any folder, for example, C drive under C:\ directory.  The eclipse will be in  C:\eclipse directory. 

Run program: C:\eclipse\eclipse.exe
